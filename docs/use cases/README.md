User -u-> Donate

Editor -r-> EditData

Admin -l-> ManageEditors

@enduml

</div>

    start

    :натискає кнопку 
       "Реєстрація";

|Система|

    :відправляє реєстраційну форму;

|Користувач|

    :вводить реєстраційні дані;

    :натискає на кнопку 
    "Зареєструватися";

|Система|

    :перевіряє отримані дані;
    note #ffaaaa
    <b> Можлива
    <b> user_registration_deny
    end note

    :створює новий обліковий запис;

|Користувач|

    :закінчує взаємодію;

    stop;
    
@enduml
